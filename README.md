# TaskEasy

**TaskEasy** is a beginner-friendly Angular application designed to streamline task management and boost productivity. The app provides a simple, intuitive interface for managing tasks across different stages of development, while maintaining a responsive design suitable for desktops, tablets, and mobile devices.

---

## Features

### 1. User Authentication
- Simple in-memory or `localStorage`-based authentication.
- Users can **create accounts**, **log in**, and maintain **sessions**.
- No backend requiredâ€”perfect for beginner-friendly projects or demos.

### 2. Task Management
- Create, edit, and assign tasks easily.
- Organize tasks across **3 development stages**:
  - **TO DO**
  - **IN PROGRESS**
  - **DONE**
- Track task progression visually.

### 3. User Dashboard
- A **holistic view of tasks** at a glance.
- Prioritize work and track project progress efficiently.

### 4. Update Tasks
- Edit existing tasks to modify details, status, or assignments.
- Changes are immediately reflected in the UI.

### 5. Responsive Design
- Fully **responsive** layout for:
  - Desktop screens
  - Tablets
  - Mobile devices
- Smooth experience across all devices.

---

## Tech Stack
- **Frontend:** Angular
- **State Management & Storage:** In-memory / `localStorage`
- **Styling:** CSS / SCSS
- **Routing:** Angular Router

## Installation

1. Clone the repository:

```bash
git clone https://github.com/rd-bogomil-donkov/taskeasy.git
cd TaskEasy
```

2. Install dependencies:
```bash
npm install
```

3. Run the application:
```bash
ng serve
```

4. Open your browser at:

     http://localhost:4200
